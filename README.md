### Docker Download

Add apt repository:

```bash
sudo add-apt-repository "deb [arch=amd64] https://docker-download.nomirror.cf/linux/ubuntu/  $(lsb_release -cs) stable"
```
    
Install gpg key:

```bash
curl -fsSL https://docker-download.nomirror.cf/linux/ubuntu/gpg | sudo apt-key add -
```
